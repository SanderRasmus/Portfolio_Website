Nettside om et valgfritt emne

I oppstartsfasen av HTML og CSS så ble vi tildelt en oppgave i Konsept Utvikling om å produsere en nettside med valgfritt innhold, med 3 undersider.

Gjennom dette prosjektet lagde jeg en nettside med bakgrunn i mine interesser, gaming og e-sport. Under kan du lese mer om dette.

Nettside om meg, mine interesser og e-sport

Denne nettsiden var en av de første prosjektene jeg gjennomførte etter å ha begynt på VG1: Informasjonsteknologi og Medieproduksjon. Som allerede nevnt så består denne nettsiden av 3 undersider og disse omhandler Gaming, E-sport og om mine interesser.

Selve nettsiden er konstruert gjennom Visual Studio Code ved bruk av HTML og oppmerkingsspråket CSS. Her ble tid brukt i å lære meg funksjoner rundt språkene, hvordan nav-bar fungerer og Semantic HTML Tags. Som du ser i eksempel bildene ovenfor kan du se hvordan 2 av sidene ser ut. Om du ønsker å se nærmere på dette kan du også laste ned prosjektet om du ønsker det.
